== Installation ==

1. Upload the plugin files to `/wp-content/plugins/smart-page-builder/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure your AI providers in Settings > Smart Page Builder
4. Enable personalization features in the Personalization tab
5. Start creating personalized content experiences!

== Frequently Asked Questions ==

= Is this plugin GDPR compliant? =

Yes! Smart Page Builder uses anonymous session-based processing with no personal data collection. All personalization is based on behavioral patterns, not personal information.

= Which AI providers are supported? =

We support OpenAI (GPT-4, GPT-3.5), Anthropic (Claude), and Google (Gemini). You can configure multiple providers and switch between them.

= How does the personalization work? =

Our Interest Vector system analyzes user behavior (clicks, time spent, scroll depth) to create mathematical profiles. These are used to personalize content in real-time without storing personal data.

= What are the performance requirements? =

The plugin is optimized for performance with sub-300ms overhead. For high-traffic sites, we recommend Redis caching support.

= Can I customize the personalization rules? =

Absolutely! The plugin provides extensive customization options through the admin interface, plus developer hooks and filters for advanced customization.

= Is there an API available? =

Yes! Smart Page Builder includes 31 REST API endpoints for external integrations, plus a webhook system for real-time notifications.
